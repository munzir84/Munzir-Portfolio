About the Project

This project uses machine learning models such as Random Forest and XGBoost to predict the Air Quality Index (AQI) for the next 7 days. It takes into account parameters like PM2.5, PM10, CO, NO₂, NH₃, SO₂, O₃, temperature, and humidity.

AQI Calculation

The AQI is calculated based on the concentration of pollutants using the formula:

AQI = ( (Observed Pollutant / Standard Limit) × 100 )

You can adjust the formula according to CPCB (India) or US EPA standards.
